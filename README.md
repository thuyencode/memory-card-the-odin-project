## Getting started

First, install all the dependencies:

```bash
npm i

# If you prefer Bun
bun i
```

To run the dev server:

```bash
npm run dev

# If you prefer Bun
bun dev
```

To export and preview the production build:

```bash
npm run build
npm run preview

# If you prefer Bun
bun run build
bun preview
```
